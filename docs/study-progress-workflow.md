# Study Progress Workflow — Revised PPT Copy

## 1. Collect Study Schemes

Gather official study schemes and organise them by programme, admission year, and stream or concentration.

## 2. Extract Rules with an AI Skill

Encode the manual interpretation framework as a reusable AI skill, then batch-analyse the schemes to extract required courses, alternative combinations, elective groups, unit thresholds, streams, and special conditions with source evidence.

## 3. Review and Program Cohort Rules

Manually verify ambiguous tables and footnotes, then use programming to convert each approved programme-year interpretation into a structured, machine-readable rule set with explicit calculation semantics.

## 4. Identify the Student Cohort

Read the student's programme, admission year, and declared stream from the transcript or profile, and match the student to the corresponding rule variant.

## 5. Map the Personal Portfolio

Match completed and planned courses to mandatory groups, elective categories, unit thresholds, and special conditions while preventing invalid double-counting.

## 6. Display Study Progress

Calculate cohort-specific completion, show satisfied and remaining requirements, and retain the matched rule source so that the result is transparent and verifiable.

## Compact Flow

**Collect Study Schemes** → **Extract Rules with an AI Skill** → **Review and Program Cohort Rules** ↓ **Display Study Progress** ← **Map the Personal Portfolio** ← **Identify the Student Cohort**
